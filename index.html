Project Overview

We were tasked with created a custom and creative D3.js project with the purpose of showing travelers and airlines where accidents occur most, and what time of the year is perhaps safer than others. Viewing these trends can provide valuable information that could help with safety precautions for both travelers and airlines. This project tested all of our capabilities and skills and pushed us all to think outside the box.

Cluster Map

This map plots the location of each aircraft crash/incident from 1962-2008 for those records that provided a latitude and longitude in the NTSB airline_accidents.csv data set. If provided in the data, the markers each specify the location, date, aircraft type, airport they originated at, and the total number of fatalities reported.

Calendar

There is a color spectrum based on the number of accidents that occur on a specific day. As you hover over each calendar square you will see details of the accidents from that day, and if you select a specific day of the calendar by clicking on it, a world tour will appear and show you the country of origin.
